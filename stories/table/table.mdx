## aui-table Inputs

| 名称       | 类型  | 默认值 | 描述       |
| ---------- | ----- | ------ | ---------- |
| dataSource | array | -      | table 数据 |

## Directives

| 名称                  | 作用范围         | 描述                                                                                  |
| --------------------- | ---------------- | ------------------------------------------------------------------------------------- |
| auiTableScrollWrapper | aui-table 的父级 | 固定列时候需要在 aui-table 的外层包裹一层，提供左右滚动效果                           |
| auiTableScrollable    | aui-table        | 表格内滚动指令，继承 cdkScrollable 指令，提供上下阴影滚动效果，固定表头时相对表格定位 |
