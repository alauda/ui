## Checkbox Group Inputs

| 名称      | 类型              | 默认值 | 描述                         |
| --------- | ----------------- | ------ | ---------------------------- |
| direction | 'row' \| 'column' | 'row'  | 按钮横纵向排列方式           |
| trackFn   | Function          | -      | 值匹配规则                   |
| value     | any               | -      | checkout group 的 value 变化 |

## Checkbox Group Outputs

| 名称        | 回调参数 | 描述                         |
| ----------- | -------- | ---------------------------- |
| valueChange | void     | checkbox group 的 value 变化 |

## Checkbox Inputs

| 名称     | 类型    | 默认值            | 描述       |
| -------- | ------- | ----------------- | ---------- |
| disabled | boolean | false             | 是否禁用   |
| label    | string  | -                 | 选项对应值 |
| value    | boolean | checkbox 的 value |

## Checkbox Outputs

| 名称        | 回调参数 | 描述                         |
| ----------- | -------- | ---------------------------- |
| valueChange | void     | checkbox group 的 value 变化 |
