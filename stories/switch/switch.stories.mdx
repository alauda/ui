## 手动控制开关态

可以人为驱动 switch 的状态变化，实现更加精细的控制

```typescript
@Component({
  template: `
    <aui-switch
      [value]="data"
      [loading]="loading"
      (valueChange)="toggle()"
    ></aui-switch>
  `,
})
export class LoadingSwitchComponent {
  data = false;
  loading = false;

  toggle() {
    this.loading = true;
    setTimeout(() => {
      this.loading = false;
      this.data = !this.data;
    }, 500);
  }
}
```

## Switch Inputs

| 名称     | 类型    | 默认值 | 描述                  |
| -------- | ------- | ------ | --------------------- |
| value    | boolean | -      | switch value          |
| disabled | boolean | false  | 禁用 switch           |
| loading  | boolean | false  | 是否处于 loading 状态 |

## Switch Outputs

| 名称        | 回调参数 | 描述                 |
| ----------- | -------- | -------------------- |
| valueChange | void     | switch 的 value 变化 |
