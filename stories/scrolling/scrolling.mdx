## 基础 API

| 名称                        | 类型      | 描述                       |
| --------------------------- | --------- | -------------------------- |
| aui-virtual-scroll-viewport | Component | 虚拟滚动容器               |
| itemSize                    | Input     | 单个内容区域高度           |
| scrolledIndexChange         | Output    | 视窗内第一个元素变化时发射 |
| \*auiVirtualFor             | Directive | 替代 \*ngFor               |
