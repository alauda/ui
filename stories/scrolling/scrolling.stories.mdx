# Scrolling

虚拟滚动模块，继承自 [CdkScrolling](https://material.angular.io/cdk/scrolling/api) ，大数据量列表使用

## 基本用法

<Canvas>
  <Story name="basic" height="280px">
    {{
      template: /* HTML */ `
        <aui-form-item>
          <label auiFormItemLabel>启用虚拟滚动</label>
          <aui-radio-group [(ngModel)]="enabled">
            <aui-radio-button [value]="true">是</aui-radio-button>
            <aui-radio-button [value]="false">
              否（点击生效前会很卡）
            </aui-radio-button>
          </aui-radio-group>
        </aui-form-item>
        <aui-virtual-scroll-viewport
          *ngIf="enabled"
          itemSize="50"
          class="example-viewport"
        >
          <div *auiVirtualFor="let item of items" class="example-item">
            {{ item }}
          </div>
        </aui-virtual-scroll-viewport>
        <div *ngIf="!enabled" class="example-viewport">
          <div *ngFor="let item of items" class="example-item">{{ item }}</div>
        </div>
      `,
      styles: [
        /* CSS */ `
          .example-viewport {
            height: 200px;
            border: 1px solid black;
            overflow-y: auto;
          }
          .example-item {
            height: 50px;
          }
        `,
      ],
      props: {
        items: Array.from({ length: 100000 }).map((_, i) => `Item #${i}`),
        enabled: true,
      },
    }}
  </Story>
</Canvas>

## 基础 API

| 名称                        | 类型      | 描述                       |
| --------------------------- | --------- | -------------------------- |
| aui-virtual-scroll-viewport | Component | 虚拟滚动容器               |
| itemSize                    | Input     | 单个内容区域高度           |
| scrolledIndexChange         | Output    | 视窗内第一个元素变化时发射 |
| \*auiVirtualFor             | Directive | 替代 \*ngFor               |
