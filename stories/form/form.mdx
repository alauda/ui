## aui-form Inputs

| 名称                 | 类型                       | 默认值 | 描述                   |
| -------------------- | -------------------------- | ------ | ---------------------- |
| auiFormLabelWidth    | string                     | -      | label 宽度             |
| auiFormLabelPosition | 'left' \| 'top' \| 'right' | -      | label 位置             |
| auiFormInline        | boolean                    | -      | 表单结构是否为内联结构 |
| auiFormEmptyAddon    | boolean                    | -      | 是否图标占位           |

## aui-form-item Inputs

| 名称  | 类型                           | 默认值 | 描述                       |
| ----- | ------------------------------ | ------ | -------------------------- |
| width | 'large' \| 'medium' \| 'small' | -      | item 的 content 宽度       |
| plain | boolean                        | false  | control 为纯文本时缩小行高 |
